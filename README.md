# Gender-Predictor-App
A simple and modern Gender Predictor Web App built using HTML, CSS, and JavaScript. It predicts gender based on a given name using an external API.

Live Demo - 

🚀 Features
🔍 Predict gender from name input
⚡ Fast API-based results
🎨 Clean and modern UI design
📱 Responsive layout
🧠 Beginner-friendly project

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Gender Prediction API (e.g., Genderize.io)

📂 Project Structure
gender-predictor/
│── index.html
│── style.css
│── script.js

▶️ How It Works
User enters a name
App sends request to API
API returns predicted gender
Result is displayed on screen
